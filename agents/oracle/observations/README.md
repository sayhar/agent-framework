# Oracle Observations

Subagent oracles write findings here during code reviews.

## File Naming

`{YYYY-MM-DD}-{short-id}.md` (e.g., `2026-01-11-auth-review.md`)

## Format

```markdown
# Review: {short description}

**Date:** YYYY-MM-DD
**Reviewed:** {what was reviewed}
**Verdict:** APPROVED | APPROVED WITH NOTES | NEEDS CHANGES | BLOCKED

## Summary

{1-3 bullet points}

## Issues

{If any issues found}

## Notes

{Additional observations}
```

## Cleanup

Delete observation files after they've been addressed and any learnings have been recorded in `../learnings.md`.
